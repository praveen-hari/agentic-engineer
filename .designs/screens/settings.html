Recovering Compacted History

When history is compacted, full artifacts are removed from the working tree but remain in git history forever. To recover:

# Find the commit that archived the work request
git log --all -- .codestudio/archive/2026-01/user-auth/
# Restore full artifacts from that commit
git checkout <commit-sha> -- .codestudio/archive/2026-01/user-auth/
# Or view a specific file without restoring
git show <commit-sha>:.codestudio/archive/2026-01/user-auth/spec.md

The extension can also do this for you — click "Restore Full Artifacts" on any compacted history entry.
