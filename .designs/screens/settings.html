Git Recovery
All workflow state is committed to git. If anything goes wrong, you can recover:
# View workflow history
git log --oneline -- .codestudio/
# Restore a previous workflow state
git checkout <commit> -- .codestudio/
# Restore compacted history entry
git show <commit>:.codestudio/history/<id>/
