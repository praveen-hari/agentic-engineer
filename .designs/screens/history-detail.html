Key Decisions
Chose Stripe over PayPal
Better developer experience, lower fees for our volume, superior documentation. See ADR-005.
Amounts in cents (integer), not dollars (float)
Per Stripe docs — avoids floating point precision issues in financial calculations.
Client-side card collection via Stripe.js (no server-side card data)
PCI compliance — card data never touches our server. Reduces security scope significantly.
